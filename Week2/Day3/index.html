What is HTML?

+

HTML stands for HyperText Markup Language. It is used to structure web pages using elements like headings, paragraphs, images and links.

What is CSS used for?

+

CSS is used for styling websites. It controls layout, colors, fonts, spacing and responsive design to make websites look beautiful.

Why do we use JavaScript?

+

JavaScript adds interactivity to websites like buttons, sliders, modals, accordions, form validation and dynamic content updates.

What is DOM Manipulation?

+

DOM Manipulation means selecting and changing HTML elements using JavaScript. We can update text, styles, or add/remove elements dynamically.
